# Changelog

## v1.0.0

Initial public deployment release.

### Added

- Topic-aware routing
- Decision support
- Cognitive distortion detection
- Workplace guidance
- Relationship guidance
- Business guidance
- Internet search through Tavily
- Research retrieval through Tavily academic-domain search
- Response critic system
- Mixed complex life problem routing
- Offline casual chat fallback
- Deterministic safety routing and audit report
- Streamlit deployment configuration

### Notes

- No fine-tuning is performed in this release.
- OpenAI is optional. Core routing, safety, decision support, emotional support,
  internet search, and research retrieval remain available without an OpenAI key.
